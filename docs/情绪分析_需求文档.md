# 情绪分析页面 - 需求文档

**版本**: v1.0
**日期**: 2025-12-12
**状态**: 待开发

---

## 一、页面定位

**核心目标**：分析连板涨停情况，判断情绪周期阶段，评估龙头股次日风险与机会

**解决的问题**：
1. 当前情绪处于什么阶段？（冰点/回暖/加速/高潮/退潮）
2. 昨日涨停股今日表现如何？（赚钱效应/大面情况）
3. 哪些概念板块有完整梯队？（主流方向）
4. 龙头股次日是机会还是风险？（操作决策）

**目标用户**：短线交易者，关注连板股接力机会

## 三、模块详细设计

### 模块1：情绪周期仪表盘

#### 1.1 展示内容

```
┌─────────────────────────────────────────────────────────────┐
│  📊 情绪周期仪表盘                          情绪阶段: 加速期 │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  ┌──────────┐  ┌──────────┐  ┌──────────┐  ┌──────────┐   │
│  │ 空间板    │  │ 晋级率   │  │ 炸板率   │  │ 涨停数   │   │
│  │    6     │  │  42.5%   │  │  28.3%   │  │   53     │   │
│  │   连板    │  │ 整体     │  │  健康    │  │  ↓-21   │   │
│  └──────────┘  └──────────┘  └──────────┘  └──────────┘   │
│                                                             │
│  晋级率明细:                                                │
│  首板→2板: 38% (8/21) | 2板→3板: 50% (4/8) | 3板→4板: 67%  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

#### 1.2 数据字段

| 字段 | 计算方式 | 说明 |
|-----|---------|------|
| 空间板高度 | 当日最高连板数 | 反映市场高度 |
| 整体晋级率 | 今日连板总数 / 昨日涨停总数 | 整体接力意愿 |
| 分层晋级率 | 今日N板数 / 昨日(N-1)板数 | 各层级晋级情况 |
| 炸板率 | 炸板数 / 涨停数 | 已有数据 |
| 涨停数 | 当日涨停总数 | 已有数据 |
| 情绪阶段 | 综合判断 | 见下方规则 |

#### 1.3 情绪周期阶段判断规则

| 阶段 | 判断条件 | 颜色 |
|-----|---------|------|
| 冰点期 | 空间板≤3板 且 炸板率>50% | 蓝色 |
| 回暖期 | 空间板=4板 或 晋级率开始回升 | 黄色 |
| 加速期 | 空间板=5-6板 且 晋级率>40% | 橙色 |
| 高潮期 | 空间板≥7板 且 涨停数多 | 红色 |
| 退潮期 | 空间板下降 或 大面率>20% | 绿色 |

#### 2.2 数据字段

| 字段 | 计算方式 | 说明 |
|-----|---------|------|
| 昨日涨停数 | 查询昨日limit_stocks_detail | 基础数据 |

| 大面率 | 情绪判断 | 操作建议 |
|-------|---------|---------|
| <10% | 情绪健康 | 可正常参与 |
| 10%-20% | 情绪偏弱 | 谨慎参与 |
| >20% | 情绪退潮 | 建议观望 |

#### 3.3 筛选逻辑

1. 筛选当日所有涨停股
2. 按概念分组，只保留有≥4板股票的概念
3. 统计每个概念在各连板层级的股票数量
4. 按最高连板数排序

#### 3.4 梯队完整度判断

| 条件 | 判断 | 说明 |
|-----|------|------|
| 涨停数≥3 且 有多层级 | ✅ 梯队完整 | 有跟风，接力氛围好 |
| 涨停数=2 或 层级单一 | 梯队一般 | 跟风不足 |
| 涨停数=1 | ⚠️ 独苗警告 | 无跟风，风险大 |

#### 4.3 数据字段

**技术面**（来自 limit_stocks_detail）：

| 字段 | 数据库字段 | 判断标准 |
|-----|-----------|---------|
| 首次封板时间 | first_limit_time | <10:00 强势 |
| 最后封板时间 | last_limit_time | - |
| 开板次数 | opening_times | 0次最强 |
| 换手率 | turnover_rate | <20% 较好 |
| 成交额 | amount | - |

**资金面**（来自 limit_stocks_detail）：

| 字段 | 数据库字段 | 判断标准 |
|-----|-----------|---------|
| 主力净流入 | main_net_inflow | >0 积极 |
| 主力净流入占比 | main_net_inflow_pct | >10% 强势 |
| 封单金额 | sealed_amount | - |
| 封单/成交比 | sealed_amount/amount | >50% 封单强 |

#### 4.4 综合评估规则

**有利因素判断**：
- 首次封板时间 < 10:00
- 开板次数 = 0
- 主力净流入 > 0
- 主力净流入占比 > 10%
- 封单/成交比 > 50%
- 有板块梯队支撑

**风险因素判断**：
- 连板数 ≥ 5（高位风险）
- 换手率 > 30%（获利盘多）
- 开板次数 ≥ 2（烂板）
- 主力净流入 < 0（资金出逃）
- 独苗无跟风

**次日预期判断**：
- 有利因素 > 风险因素 → "机会 > 风险"
- 有利因素 = 风险因素 → "机会与风险并存"
- 有利因素 < 风险因素 → "风险 > 机会"
