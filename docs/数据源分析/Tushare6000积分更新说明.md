## 🤔 概念板块：Tushare vs AKShare

虽然你现在可以用 Tushare 获取概念数据，但建议**仍然使用 AKShare**：

### Tushare 概念数据

```python
# Tushare 同花顺概念（6000积分）
df = pro.ths_index(exchange='A')

# 返回: 1234 条数据
# 类型: 地域概念（安徽、北京等）+ 行业概念
# 例如:
#   ts_code    name    count
#   882001.TI  安徽    159
#   882002.TI  北京    448
```

**特点**：
- ✅ 官方数据，稳定
- ⚠️ 主要是**地域和行业概念**
- ⚠️ 缺少热门题材概念（如"AI概念"、"ChatGPT概念"等）
- ⚠️ 获取成分股需要额外API调用

### AKShare 概念数据

```python
# AKShare 同花顺概念（免费）
df = ak.stock_board_concept_name_ths()

# 返回: 374 条数据
# 类型: 热门题材概念
# 例如:
#   板块名称      板块代码  涨跌幅
#   AI概念       308614   5.2%
#   ChatGPT概念  309121   3.8%
```

**特点**：
- ✅ **完全免费**
- ✅ **热门题材概念**（短线复盘最需要的）
- ✅ 直接返回涨跌幅
- ✅ 轻松获取成分股和龙头股
- ✅ 数据更新及时
