# A股短线复盘工具 - 二期需求文档

**版本**: v2.0
**日期**: 2025-12-10
**状态**: 待开发

---

## 一、项目背景

### 1.1 一期成果回顾

一期（v1.0 - v1.4.5）已完成：
- ✅ 数据采集模块（涨停池、热门概念、市场情绪、龙虎榜）
- ✅ 后端API（11个接口）
- ✅ 首页仪表板（大盘指数、市场情绪、热门概念TOP10、涨停池列表）
- ✅ 5维度市场情绪评分系统

**一期定位**：快速数据看板，了解当日市场概况

### 1.2 二期目标

**核心目标**：为短线交易者提供完整的**交易决策支持系统**

**决策链路**：
```
大盘分析（能不能做？） → 板块分析（往哪里做？） → 个股分析（买哪只？）
```

---

## 二、核心需求概述

### 2.1 产品定位升级

从"数据展示工具" → "交易决策系统"

**核心价值**：
1. **大盘分析** - 判断市场环境，决定是否出手、仓位多少
2. **板块分析** - 识别主线方向，找到热点机会
3. **个股分析** - 精准选股，确定买卖点

### 2.2 用户痛点

**短线交易者的3大核心问题**：

1. **市场环境判断难**
   - 情绪好不好？能不能做？
   - 应该满仓/半仓/空仓？

2. **热点方向难把握**
   - 哪些板块是主线？能延续几天？
   - 连板股哪些有机会，哪些有风险？

3. **个股选择难**
   - 买龙头还是跟风？买高位还是低位？
   - 什么价格进场？止损位在哪？

**数据输入**（传给AI）：
- 市场情绪评分（-5 到 +5）
- 涨跌比、涨停数、跌停数、炸板率
- 成交额及环比变化
- 连板梯队分布（1板X只、2板Y只...）
- 热门板块TOP5（名称、涨幅、涨停数）
- 三大指数涨跌幅、量比

**AI输出要求**（Prompt约束）：
1. 可操作性评分：1-5分（精确到0.5分）
2. 明确结论：满仓/半仓/轻仓/空仓+具体操作方向
3. 有利因素：3条，简洁具体
4. 不利因素：3条，简洁具体
5. 操作建议：仓位+方向+风险提示+交易策略
6. 字数控制：整体≤300字

**主线识别规则**（后端算法）：
1. 统计近7日各概念的涨停数总和
2. 筛选条件：
   - 累计涨停 ≥ 15只次
   - 连续上涨天数 ≥ 3天
   - 有明确龙头（连板数 ≥ 2）
3. 趋势阶段判断：
   - 萌芽期：连续2-3天，涨停数递增
   - 启动期：连续3-4天，涨停数持续增长
   - 加速期：连续4-7天，涨停数达到峰值
   - 衰退期：涨停数开始下降
